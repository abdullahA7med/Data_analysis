## 🔍 Project Highlights

### ✨ Exploratory Data Analysis (EDA)
- Univariate & multivariate analysis
- Missing value detection & handling
- Outlier identification using IQR & Z-score
- Correlation heatmaps & pair plots

### 🧹 Data Preprocessing
- Feature scaling (StandardScaler, MinMaxScaler,..)
- Encoding categorical variables (One-Hot, Label)
- Train-test splitting & stratification
- Pipeline integration for reproducibility

### 📊 Interactive Dashboards
- Streamlit apps for dynamic filtering & visualization
- Plotly interactive charts for stakeholder presentations
